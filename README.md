### Install
Run this command:
```bash
git clone --depth 1 https://github.com/klinkonskydev/neovim ~/.config/nvim
```

Remove our git:
```bash
rm -rf ~/.config/nvim/.git
```

Run neovim in your terminal:
```bash
nvim
```
